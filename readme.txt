This plugin lets you place a Facebook Like button on your WordPress blog as a widget. You can configure the button by choosing the style and opting to show faces.

== Description ==

To use the widget just drag it to a sidebar and choose one of three button styles. You can also choose to show faces, faces will then be shown when using the standard button style. 

== Installation ==

Upload the Facebook Like Social Widget plugin to your blog, activate it and drag the widget to your sidebar.
